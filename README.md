## 🔑 Έλεγχος Βαθμού Μαθητή

**Βαθμός μαθητή (Όριο):** 72

Ο παραπάνω βαθμός χρησιμοποιείται ως όριο για την χρωματική μορφοποίηση των βάσεων εισαγωγής στον πίνακα:

* **Πράσινο:** Αν η Βάση Εισαγωγής είναι **μικρότερη ή ίση** με τον Βαθμό Μαθητή ($\le 72$).
* **Κόκκινο:** Αν η Βάση Εισαγωγής είναι **μεγαλύτερη** από τον Βαθμό Μαθητή ($> 72$).
